# Scrapy-mts

NPSN islamic junior high school Scrapper

## Change base url

```bash
base_url = 'change according to your needs'
}
```

## Change sub-base url

'sub-base-url' is the url per sub-district of 'base-url'

```bash
if i < 10:
        url = f'sub-base_url 1'
    else:
        url = f'sub-base_url 2'
}
```
